### zero attributes:

```js
;<MobilePageStyleguidist.Component>
    <Title>
        This is a default Title component with no
        attributes whatsoever.
    </Title>
</MobilePageStyleguidist.Component>
```

### type:

```js
;<MobilePageStyleguidist.Component>
    <Title type="primary">type - primary</Title>
    <Title type="secondary">type - secondary</Title>
    <Title type="success">type - success</Title>
    <Title type="warning">type - warning</Title>
    <Title type="danger">type - danger</Title>
</MobilePageStyleguidist.Component>
```

;<MobilePageStyleguidist.Component>

    <Title size="sm">size - sm - small</Title>

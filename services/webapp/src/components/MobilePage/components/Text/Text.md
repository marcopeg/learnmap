### zero attributes:

```js
;<MobilePageStyleguidist.Component>
    <Text>
        This is a default text component with no
        attributes whatsoever.
    </Text>
</MobilePageStyleguidist.Component>
```

### type:

```js
;<MobilePageStyleguidist.Component>
    <Text type="primary">type - primary</Text>
    <Text type="secondary">type - secondary</Text>
    <Text type="success">type - success</Text>
    <Text type="warning">type - warning</Text>
    <Text type="danger">type - danger</Text>
</MobilePageStyleguidist.Component>
```

### size:

```js
;<MobilePageStyleguidist.Component>
    <Text size="xs">size - xs - extra small</Text>
    <Text size="sm">size - sm - small</Text>
    <Text size="md">size - md - medium (default)</Text>
    <Text size="lg">size - lg - large</Text>
    <Text size="xl">size - xl - extra large</Text>
</MobilePageStyleguidist.Component>
```
